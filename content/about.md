i am spiz. Spirit, Power, Intimacy, Zeal — JustPassingBy.

I build Agentic AI Systems.

Not because I studied computer science. I built them because I got tired of repeating the same instructions to every new AI tool that came alongi...because i had to try them all. One fix turned into another. Then another. I wanted everything up to date wiht the same skill/knowledge...etc. One ring to bind them. Eventually the fixes and migrations connected to each other and I realized I had accidentally built an architecture.

What emerged is a small AI execution system: a central orchestrator (the brain), specialized agents that do specific work, shared knowledge so agents do not relearn the same things, persistent memory so context survives between runs, and a ledger that records what happened so the system can recover, retry, or continue without starting over.

Instead of one giant prompt trying to do everything, the work moves through pipelines. Research agents find opportunities. Design agents generate images. Writing agents create listings. Validators check outputs. Publishing agents ship the results.

The whole thing runs on a tiny AWS server that costs roughly ten dollars a month. It researches trends, generates designs, publishes products to Etsy, monitors financial signals, and runs a few other bot-driven experiments. I originally built it to solve practical problems. The architecture itself turned out to be the interesting part.

This blog is where I write about what I am building, what breaks, what evolves, and what it is like to be a solo operator turning AI from a chat tool into a system that does real work.

I think in songs, visions, scenes, and scripture. I build in Node.js, Python, and midnight debugging sessions.

Both are true at the same time.

This is AI Tinkers.

Notes from building strange systems.
